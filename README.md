# 계산기 (Calculator)

정수 및 소수점 연산을 지원하는 현대적인 웹 계산기입니다.

![Calculator Screenshot](screenshot.png)

## ✨ 주요 기능

### 기본 기능
- ✅ **숫자 입력**: 0-9 버튼으로 최대 8자리까지 입력 가능
- ✅ **기본 연산**: 덧셈(+), 뺄셈(-), 곱셈(×), 나눗셈(÷) 지원
- ✅ **계산 실행**: = 버튼으로 결과 확인
- ✅ **지우기 기능**:
  - **C**: 마지막 입력 지우기
  - **AC**: 전체 초기화
- ✅ **오류 처리**: 8자리 초과 시 'ERR' 메시지 표시

### 보너스 기능
- 🎁 **부호 변경**: +/- 버튼으로 양수/음수 전환
- 🎁 **소수점 연산**: 최대 소수점 3자리까지 지원

### 추가 기능
- ⌨️ **키보드 지원**: 숫자, 연산자, Enter(=), Escape(AC), Backspace(C) 키 사용 가능
- 📊 **연산 히스토리**: 이전 연산 과정 표시
- 📱 **반응형 디자인**: 모바일, 태블릿, 데스크톱 모두 지원

## 🎨 디자인 특징

- **다크 테마**: 눈의 피로를 줄이는 현대적인 다크 모드
- **그라데이션 컬러**: 생동감 있는 보라-핑크 색상 팔레트
- **애니메이션**: 부드러운 호버 효과와 마이크로 인터랙션
- **글래스모피즘**: 반투명 효과로 깊이감 표현
- **네온 효과**: 발광하는 텍스트와 버튼 효과

## 🚀 사용 방법

### 실행하기
1. `index.html` 파일을 브라우저에서 열기
2. 또는 라이브 서버 사용:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js
   npx serve
   ```

### 계산 예시
```
12 + 34 = 46
100 - 25 = 75
15 × 3 = 45
50 ÷ 2 = 25
12.5 + 7.3 = 19.8
```

### 키보드 단축키
| 키 | 기능 |
|---|---|
| `0-9` | 숫자 입력 |
| `+` `-` `*` `/` | 연산자 |
| `Enter` 또는 `=` | 계산 실행 |
| `Escape` | 전체 초기화 (AC) |
| `Backspace` | 지우기 (C) |
| `.` | 소수점 |

## 📝 구현 세부사항

### 입력 제한
- 정수 부분: 최대 8자리
- 소수 부분: 최대 3자리
- 8자리 초과 결과: 'ERR' 표시

### 연산 로직
- 연속 연산 지원 (예: 1 + 2 + 3 + 4)
- 0으로 나누기 방지
- 소수점 반올림 처리

### 상태 관리
- 현재 값 (currentValue)
- 이전 값 (previousValue)
- 선택된 연산자 (operator)
- 새 값 대기 상태 (waitingForNewValue)
- 오류 상태 (hasError)
